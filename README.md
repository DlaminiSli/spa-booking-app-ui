# BrightSpa Mobile App UI Design for Appointment Booking

## Project Overview
This project is a mobile app UI design for a spa and wellness business, allowing users to easily book appointments.

The goal is to:
- Simplify the booking process
- Reduce reliance on calls and messages
- Improve customer experience
- Provide a smooth mobile-first interaction

---

## Features
- User onboarding and login
- Service selection
- Specialist selection
- Date and time booking
- Booking summary
- Appointment confirmation

---

## Booking Flow
1. User logs in  
2. Selects a service  
3. Views service details  
4. Selects specialist  
5. Chooses date and time  
6. Confirms booking
